## Activity 3 : 결과 공유

---

### 어떤 AI를 만들었나요?
- 티처블 머신을 활용한 이미지 분류 AI
- 두 가지 모델 제작
  - 포켓몬 분류: 불타입 / 물타입 / 풀타입
  - 과일 분류: 사과 / 바나나 / 오렌지

---

### 정확도 비교
- 포켓몬 AI: 높은 정확도
  - 색과 특징이 뚜렷하여 구분이 쉬움
- 과일 AI: 상대적으로 낮은 정확도
  - 색과 형태가 유사하여 혼동 발생

---

### 오답이 발생한 경우
- 배경이 복잡한 이미지
- 색이 비슷한 경우 (사과 vs 오렌지)
- 형태가 변형된 경우 (잘린 과일 등)
- 학습 데이터 부족

---

## 2. 토론 질문

### Q1. 이 AI가 지도학습인 이유는?
- 이미지와 정답 라벨을 함께 학습했기 때문
- 입력 데이터와 정답의 관계를 학습하는 방식

---

### Q2. 포켓몬 AI가 더 정확한 이유
- 색과 특징이 뚜렷함
- 클래스 간 차이가 큼
- 배경이 단순한 데이터가 많음

---

### Q3. 실제 마트에서 활용하려면?
- 더 많은 데이터 수집
- 다양한 환경(조명, 각도) 고려
- 배경 제거 및 객체 인식 적용
- 품종별 세분화
- 실시간 처리 시스템 구축
